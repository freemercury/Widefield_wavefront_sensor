# 运行流程
0.  batch_realign/batch_usage_0407.m, batch_usage_0408.m：处理230406和230407数据，这里时间标错位了ww，生成realign后的文件
1.  重命名用advanced renamer，去噪用denoise.ijm处理，生成realign后的文件
2.  运行mask_and_linear_proj.m，生成文件S2Zmatrix.mat，包含projection matrix和mask
3.  运行img2zernike_230407.py，设置SAVE_SHIFTMAP=True, SAVE_ZERNIKE=False, 生成shiftmap.mat
4.  运行shiftmap2zernike_230407.m，生成zernike_full.mat
5.  运行mlp_proj.py，设置TRAIN=True，训练生成mlp_model.pkl，也可快速评估模型在测试集上的性能
6.  运行img2zernike_230407.py，设置SAVE_SHIFTMAP=False, SAVE_ZERNIKE=True，生成zernike.mat
7.  运行zernike_prepare.py，整合zernike生成pred_data，用于预测训练
8.  运行img2zernike_speed_test.py，用于测试算法速度

# 数据说明
1.  shiftmap.mat: ["shiftmap"] size of (15,15,19,25,2)
2.  S2Zmatrix.mat: ["mask"] size of (15,15), ["S2Zmatrix"] size of (280,34)
3.  zernike.mat: ["zernike"] size of (35,19,25)
4.  zernike_full.mat: ["zernike_full"] size of (35,19,25)
